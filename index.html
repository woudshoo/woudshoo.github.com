Common Lisp Projects

Graph Related

This project is my take on graph reductions. The projects are mainly inspired by my desire to make sense of reivision graphs gone mad. The projects try to give a different view on the revision graph by trying to distill a smaller graph still retaining all the key features of the original graph. Or make a smaller graph by zooming in on a small subset of revisions.
WO Graph documentation
This contains the basic infrastructure, representing a graph. It is just infrastructure and defines an interface for the algorithms on top. By implementing this interface you can define your own graph datastructure for usage by the algorithms in the next project.
WO Graph functions documentation
This project contains graph algorithms. The graphs should implement the interface defined in the project above.
CL Git, libgit2 bindings documentation
This project is forked/collaborated from Russell Sim's cl-git project. It provides bindings to manipulate git repositories.
